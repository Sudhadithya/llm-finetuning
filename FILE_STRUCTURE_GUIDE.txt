STEP-BY-STEP SETUP
==================

1. CREATE PROJECT FOLDER
   mkdir llm-finetuning
   cd llm-finetuning

2. CREATE FOLDER STRUCTURE
   mkdir -p data/{raw,processed}
   mkdir -p training evaluation inference models configs docker notebooks outputs/{checkpoints,logs,results}

3. DOWNLOAD FILES FROM OUTPUTS
   Download these 6 files into your llm-finetuning folder:
   
   ✓ dataset_builder.py        → training/
   ✓ train_lora_optimized.py   → training/ (rename to train_lora.py)
   ✓ training_config.yaml      → configs/
   ✓ .gitignore                → (root)
   ✓ README.md                 → (root)
   ✓ SETUP_CHECKLIST.md        → (root)

4. CREATE requirements.txt (or download if provided)
   Create file: requirements.txt with this content:
   
   torch>=2.1.0
   transformers>=4.38.0
   datasets>=2.17.0
   accelerate>=0.27.0
   peft>=0.8.2
   bitsandbytes>=0.42.0
   sentencepiece>=0.1.99
   tokenizers>=0.15.0
   evaluate>=0.4.1
   sacrebleu>=2.4.0
   bert-score>=0.3.13
   rouge-score>=0.1.2
   numpy>=1.24
   pandas>=2.0
   fastapi>=0.110.0
   uvicorn>=0.27.0
   pydantic>=2.6.0
   tqdm>=4.66
   pyyaml>=6.0
   scikit-learn>=1.3
   wandb>=0.16.0
   matplotlib>=3.7
   jupyter>=1.0

5. ACTIVATE VIRTUAL ENVIRONMENT
   .\llm-env\Scripts\activate  (Windows)
   source llm-env/bin/activate (Mac/Linux)

6. INSTALL DEPENDENCIES
   pip install -r requirements.txt

7. RUN DATASET BUILDER
   python training/dataset_builder.py
   This creates: data/processed/train.json

8. RUN TRAINING
   python training/train_lora.py
   This trains the model and saves to: models/phi2-lora/

FILE DESCRIPTIONS
=================

dataset_builder.py
├─ Purpose: Downloads Databricks Dolly dataset, processes it
├─ Input: (downloads from HuggingFace)
├─ Output: data/processed/train.json
├─ Runtime: ~5-10 minutes (depends on internet)
└─ Run: python training/dataset_builder.py

train_lora.py (renamed from train_lora_optimized.py)
├─ Purpose: Main training script with LoRA fine-tuning
├─ Input: data/processed/train.json
├─ Output: 
│   ├─ models/phi2-lora/
│   ├─ outputs/checkpoints/
│   └─ outputs/logs/
├─ Runtime: ~2-3 hours on RTX 4060
├─ GPU Memory: 7.8GB / 8GB
└─ Run: python training/train_lora.py

training_config.yaml
├─ Purpose: Configuration file for training hyperparameters
├─ Edit this to adjust:
│   ├─ batch_size
│   ├─ learning_rate
│   ├─ num_epochs
│   └─ lora rank/alpha
└─ Used by: train_lora.py (you can reference it)

requirements.txt
├─ Purpose: Lists all Python dependencies
├─ Used by: pip install -r requirements.txt
└─ Contains: PyTorch, Transformers, PEFT, etc.

.gitignore
├─ Purpose: Tells Git which files to ignore
├─ Ignores: /__pycache__, /models, /data, /outputs, /llm-env
└─ Used by: git (if you push to GitHub)

README.md
├─ Purpose: Full project documentation
├─ Contains: Architecture, setup, usage, troubleshooting
└─ Read this for detailed information

SETUP_CHECKLIST.md
├─ Purpose: Step-by-step setup verification checklist
├─ Contains: All steps needed to get running
└─ Use this to verify your setup is correct
